# HRMS Lite

HRMS Lite is a lightweight Human Resource Management System built as part of a full-stack development assessment.  
The application allows an admin to manage employees and track daily attendance through a simple and clean interface.

## Features

### Employee Management
- Add employee
- View employees
- Delete employee (soft delete)
- Validation for duplicate employee IDs
- Email format validation

Fields:
- Employee ID
- Full Name
- Email Address
- Department

---

### Attendance Management

- Mark attendance
- View attendance per employee
- Attendance filter by employee and date

Fields:
- Date
- Status (Present / Absent)
